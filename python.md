## virtualenvwrapper

### Installation

```
pip install virtualenvwrapper
echo 'source /home/calimaborges/.local/bin/virtualenvwrapper.sh' > ~/.bashrc
```

## pip

### install packages

```
pip install <package>
```

### freeze requirements

```
pip freeze > requirements.txt
```

### install requirements

```
pip install -r requirements.txt
```
