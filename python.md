# Python

## virtualenvwrapper

### installation

```
pip install virtualenvwrapper
echo 'source /home/calimaborges/.local/bin/virtualenvwrapper.sh' > ~/.bashrc
```

### list virtual envs

```
lsvirtualenv
```

### create virtual env

```
mkvirtualenv <name>
```

### work on virtual env

```
workon <name>
```

## pip

### install packages

```
pip install <package>
```

### freeze requirements

```
pip freeze > requirements.txt
```

### install requirements

```
pip install -r requirements.txt
```
